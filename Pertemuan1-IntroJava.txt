1. Jelaskan apa itu Jre ? Apa kegunaannya!
JRE atau singkatan dari Java Runtime Environment adalah teknologi dasar yang melakukan komunikasi antara program Java dan sistem operasi.

2. Jelaskan apa itu JDK?
JRE bertindak sebagai penerjemah dan fasilitator, menyediakan semua sumber daya sehingga setelah Anda menulis perangkat lunak Java, JRE berjalan pada sistem operasi apa pun tanpa modifikasi lebih lanjut.

3. Jelaskan apa itu Java Virtual Machine?
Java Virtual Machine atau disingkat JVM merupakan mesin virtual yang digunakan secara khusus mengeksekusi berkas bytecode java.
